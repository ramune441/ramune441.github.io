Features

できること

足あとメーカーが提供する主な機能をご紹介します。

👁

プロフィール自動訪問

対応アプリのページを開いてボタンを押すだけ。プロフィールを自動で巡回し、相手に足あとを残します。

🔄

無限ループモード

ページを自動リロードして新しいプロフィールを読み込み、訪問を繰り返します。放置するだけでOK。

❤️

自動いいね機能

東カレデート・Tinderでは、訪問と同時に自動で「いいね」を送ることも可能です。

⏱

訪問間隔の調整

訪問ごとの待機時間を自由に設定。自然なペースでの訪問が可能です。

🔢

訪問上限の設定

1回あたり・全体の訪問回数に上限を設定できます。使いすぎを防止。

🔧

カスタム対応

URLパターンとCSSセレクターを指定すれば、プリセット以外のサイトにも対応できます。

How to use

使い方はかんたん3ステップ

インストールから自動訪問開始まで、わずか数分で完了します。

Chromeに追加

Chrome ウェブストアから「足あとメーカー」をインストールします。登録やログインは不要です。

アプリを開く

対応するマッチングアプリのWebページをChromeで開きます。操作パネルが右下に自動表示されます。

開始ボタンを押す

「開始」ボタンを押すだけで自動訪問がスタート。あとは放置するだけです。
